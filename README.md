# GoTogo – Plateforme de Covoiturage en PHP (MVC)

GoTogo est une application web de covoiturage développée en PHP avec une architecture MVC.  
Elle permet aux utilisateurs de créer un compte, proposer des trajets, réserver des places et gérer leurs réservations.

## 🚀 Fonctionnalités principales

- Création de compte et connexion sécurisée (hashage des mots de passe)
- Gestion des rôles : client / conducteur
- Création et publication de trajets
- Réservation d’un trajet par un client
- Gestion du statut des réservations (en attente, acceptée, refusée, annulée)
- Tableau de bord utilisateur
- Base de données MySQL structurée avec clés étrangères

## 🛠️ Technologies utilisées

- **PHP 8+**
- **MySQL**
- **HTML / CSS**
- **WAMP / XAMPP**
- **Architecture MVC**
